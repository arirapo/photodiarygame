Photodiarygame

Choose a way of seeing.

Photodiarygame is a small web-based environment for image, attention, memory, and poetic response. This gallery page works as a quiet entry point to two different modes: a chain-based image experience and a Miksang-inspired practice flow.

The first module continues the original Photodiarygame logic: image → poem → response. The second module slows the process down through three stages: Pause → Notice → Capture.
Module 01

Photodiarygame

The original chain-based experience. A photograph becomes a poetic prompt and opens a path for the next image. This mode feels more playful, generative, and connective.
