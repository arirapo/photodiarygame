# The Face Between

Basic ghost-base prototype for the `photodiarygame/TheFaceBetween/` project.

## Files
- `index.html` – page structure
- `style.css` – visual layers and prototype styling
- `app.js` – demo fragments, prompt rotation, ghost opacity logic
- `assets/ghost-base.jpg` – starting ghost image

## Current state
This version is intentionally lightweight:
- shows the ghost base
- shows demo mosaic fragments
- rotates a prompt
- calculates ghost opacity from active trace count

## Not connected yet
- Firebase
- real upload flow
- image analysis
- Firestore / Storage
